# 🏥 Hospital Management System (Java + MySQL)

## 📌 Project Description

The Hospital Management System is a console-based Java application that helps manage patients, doctors, and appointment bookings.
It is developed using **Core Java, JDBC, and MySQL** to demonstrate database connectivity and CRUD operations.

---

## 🚀 Features

* ✅ Add new patients
* ✅ View patient details
* ✅ View doctor list
* ✅ Book appointments
* ✅ Check doctor availability
* ✅ Uses PreparedStatement for secure queries
* ✅ Follows modular OOP structure

---

## 🛠 Technologies Used

* Java (Core Java)
* JDBC
* MySQL Database
* Eclipse IDE

---

## 🗄 Database Tables

* Patients
* Doctors
* Appointments

SQL file included in repository:

```
hospital_database.sql
```

---

## ▶ How to Run the Project

1. Import project into Eclipse
2. Run MySQL server
3. Execute `hospital_database.sql` in MySQL
4. Update DB username/password in code if needed
5. Run `HospitalManagementSystem.java`
